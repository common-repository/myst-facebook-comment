Replace Wordpress Comment Template into Facebook Comment Plugin

== Description ==

This plugin offers simple setup for your Facebook comment plugin integration by replacing every comment_template() located in single.php, page.php, and image.php into Facebook Comments Plugin.

NOTE : 
**This plugin only been tested under Wordpress default template.**

Do let us know what other features that we should have, and we shall research through if it is possible to implement it.

== Installation ==

Installing "MYST Facebook Comment" can be done either by searching for "MYST Facebook Comment" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
